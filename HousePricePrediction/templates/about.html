About this project

This project is about:

  • Developing a house price prediction system using machine learning.
  • Utilizing Linear Regression for accurate price prediction based on various factors.
  • Building the web user interface using Django, a high-level Python web framework.
  • Training the model using scikit-learn, a popular machine learning library in Python.
